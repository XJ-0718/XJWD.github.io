##XJWD.github.io#XJWD.github.io
这是基于GitHub页面搭建的个人静态站点仓库。

##项目介绍
本仓库用于存放个人静态网页的源码，通过GitHub页面服务实现网页的在线访问。

##访问地址
-在线访问：[https://XJWD.github.io](https://XJWD.github.io)

##技术栈
-HTML/CSS/JavaScript
-降价（文档编写）
-GitHub页面（部署服务）

##部署说明
1.将静态网页文件（如`index.html`）直接推送到本仓库的`主要的`分支
2.开启GitHub页面功能，选择`主要的`分支作为源
3.访问`https://XJWD.github.io`即可查看站点
